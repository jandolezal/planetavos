Nauč se programovat

21. února 2021

Každej by prej měl umět programovat. Nee. Každej by prej měl umět myslet. Jakože logicky. STEM rulez! A tak. Takže kde začít? Dneska asi na Junior Guru.

Začít s JavaScriptem bylo hození do vody až moc z vysoka. Nejvíc pomohl úvod do programování a Python na edX a knížky jako třeba Think Python. Spíš než do Automate the boring stuff with Python pořád koukám do Introducing Python.

Jenže Python nestačí. Takže SQL, takže HTML, CSS. Potřebujeme Django, ale lepší je začít s Flaskem.
